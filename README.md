# Casey Hartman - Personal Website

## Overview

This repository contains the source code for my personal website.

The site presents my academic background in astrophysics, ongoing research projects, and personal interests including music and astrophotography.

Live site: [chartman25.github.io](https://chartman25.github.io)

---

## Structure

- `index.html` — Homepage with bio and recent news
- `research.html` — Research projects (current and past)
- `music.html` — Music projects and interests
- `contact.html` — Contact form and links
- `cv.pdf` — Curriculum Vitae
- `photo.jpg` — Profile photo
- `css/style.css` — Shared styles

---

## Tech Stack

- HTML and CSS for structure and styling
- [Formspree](https://formspree.io) for contact form handling
- GitHub Pages for hosting

---

## Deployment

The site is deployed using GitHub Pages from the `main` branch.

---

## Features

- Minimal academic design with purple accent color matching CV style
- Research project cards with links to GitHub/GitLab repositories
- Contact form with email delivery via Formspree
- CV accessible as a direct PDF download
- Responsive layout for desktop and mobile
